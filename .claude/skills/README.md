# Skills del proyecto

Índice de los skills en `.claude/skills/`. Cada uno tiene su propio `SKILL.md`
con el detalle completo; esto es solo para ubicarlos rápido.

- **conventional-commit** — Genera mensajes de commit siguiendo Conventional
  Commits. Se usa al crear un commit o cuando se pide un mensaje de commit.
- **create-prd** — Crea o audita un PRD siguiendo el template y el checklist
  de calidad del curso, iterando en loop hasta que quede firme. Se usa cuando
  se pide crear, escribir, revisar, auditar o endurecer un PRD.
